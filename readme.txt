Displays your photo stream from your Frin.ly account

== Description ==

Share photos on Twitter. Frin.ly is a free online photo sharing site that allows users to share your photos on twitter.
This widget will display your latest Frinly photos. It adds a sidebar widget which will display your latest Frinly photo stream. 

== Installation ==

1.  Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

2.  Set the 3 options that are given.

a). Title (Joe's Photo's), 
b). Frin.ly Username (http://frin.ly/username)
c). Images displayed (How many images would you like to show)
